# 陈叔叔工具箱 更新日志

## 新增功能

- **统一配置管理系统** - 使用 `config.json` 统一管理工具和分类

- **添加工具功能** - 标题栏"+"按钮，支持添加文件或文件夹到指定分类
- **删除工具功能** - 右键工具卡片可删除工具
- **删除分类功能** - 右键导航栏分类可删除分类
- **平滑滚动动画** - 工具卡片页和导航栏支持惯性滚动
- **更新检查气泡** - 启动时检查更新，在卡片页右上角显示悬浮提醒
- **设置页面** - 标题栏齿轮图标，支持检查更新、自动更新开关
- **日志记录系统** - 添加/删除操作全过程记录到 `debug.log`

## 优化
- **图标缓存机制** - 首次启动自动提取工具图标到 `IconCache/`，后续直接读取缓存，极大程度上提高了低端设备打开工具箱的速度
- 展开下拉项时窗口会滚动到底部
- 合并扩展工具目录到统一的"工具"文件夹
- 导航栏和全部工具页面按指定顺序排列分类
- 对话框统一使用 Windows 原生窗口样式
- 按钮添加线性高亮动画
- 工具扫描过滤非工具文件
